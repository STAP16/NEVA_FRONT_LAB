# NEVA LAB — Брендбук

---

## 1. Позиционирование

**NEVA LAB** — образовательная лаборатория с технологическим характером и дружелюбной подачей.
Визуальный язык строится на контрасте: серьёзная экспертиза подаётся через мягкий, воздушный
интерфейс с маскотом-проводником.

**Характер интерфейса:** мягкий футуризм — glassmorphism, градиенты, blur-эффекты, плавные анимации.
Технологичность без агрессии.

---

## 2. Визуальные принципы

| #   | Принцип                             | Проявление                                                                                                                   |
| --- | ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| 1   | **Воздушность**                     | Светлые фоны `#f0f7ff`→`#ffffff`, стеклянные карточки с `backdrop-filter: blur`, минимум плотных блоков                      |
| 2   | **Монохромная глубина**             | Единая синяя палитра от `#1e3a5f` до `#7fb4e3` — без посторонних цветов. Глубина через оттенки, а не через разнородность     |
| 3   | **Плавность**                       | Все переходы через `ease` (0.2–0.7s), появление контента через `fadeUp`, scroll-driven анимации маскота                      |
| 4   | **Маскот как голос**                | NEVIK присутствует в 5 из 6 экранов, имеет речевые пузыри — это не декорация, а персонаж-рассказчик                          |
| 5   | **Стеклянная карточка как единица** | Все блоки контента оформлены как glass-card: `rgba(255,255,255,0.7)` + `blur(8px)` + `border: 1px solid rgba(44,90,160,0.1)` |

**Эмоциональный тон:** уверенный, но тёплый. Экспертность без надменности. Наука, поданная с
заботой.

**Визуальная метафора:** стекло и свет — прозрачность процессов, ясность знаний, чистота подхода.

## 4. Типографическая система

### Шрифт

**Montserrat** — единственный шрифт. Геометрический гротеск с мягким характером.

Начертания: **400** (Regular), **500** (Medium), **600** (SemiBold), **700** (Bold)

### Шкала размеров (fluid, `clamp()`)

| Уровень             | Диапазон                             | Вес     | Высота строки |
| ------------------- | ------------------------------------ | ------- | ------------- |
| **Display / H1**    | 28–52px (`clamp(28px, 4vw, 52px)`)   | 700     | 1.15          |
| **H1 CTA**          | 32–64px (`clamp(32px, 5.5vw, 64px)`) | 700     | 1.2           |
| **H2**              | 24–42px (`clamp(24px, 4vw, 42px)`)   | 600     | 1.3           |
| **H3 / Card title** | 20–26px (`clamp(20px, 2vw, 26px)`)   | 600     | 1.4           |
| **Subtitle**        | 15–20px (`clamp(15px, 2vw, 20px)`)   | 400–500 | 1.6           |
| **Body**            | 14–17px (`clamp(14px, 1.4vw, 17px)`) | 400     | 1.6           |
| **Small / Label**   | 12–14px (`clamp(12px, 1.2vw, 14px)`) | 500     | 1.4           |
| **Stat Number**     | 36–56px (`clamp(36px, 5vw, 56px)`)   | 700     | 1.15          |

### Правила

- Заголовки: `color: #1e3a6e`, вес 600–700
- Основной текст: `color: #3d5a80`, вес 400
- Подписи: `color: #5a7aa0` или `#8192b3`, вес 400–500
- Логотип: `letter-spacing: 4px`, uppercase, 700
- Лейблы секций: `letter-spacing: clamp(2px, 0.4vw, 5px)`, uppercase, 600

## 5. Маскот NEVIK — правила использования

### Общие принципы

- NEVIK — маскот-проводник, не декорация. Всегда имеет контекстную роль в секции
- Присутствует в **5 из 6 экранов** страницы «О лаборатории»
- Может иметь **речевой пузырь** (chat bubble) с репликой

### Варианты поз

| Поза                    | Файл                               | Контекст                |
| ----------------------- | ---------------------------------- | ----------------------- |
| Приветствие             | `NEVIK_FOR_ABOUT_FIRST_SCREEN.png` | Hero-секция, знакомство |
| Презентация результатов | `NEVIK_WHAT_YOU_GET.png`           | Секция «Что получите»   |
| Обращение к аудитории   | `NEVA_FORWHOM.png`                 | Секция «Кому подходит»  |
| В кепке (неформальный)  | `NEVA_CAP_1.png`                   | CTA-секция, призыв      |

### Правила размещения

- Максимальная ширина: **580px** (hero, CTA), до **1260px** (фоновый)
- Позиционирование: **абсолютное**, часто выходит за границы секции (`bottom: -60px` или `-200px`)
- На мобильных (`<768px`): **скрывается** в фоновых вариантах
- Анимация входа: `fadeIn` + `slideFromRight` (x: 40→0) или scroll-driven (y: 500→0)

### Речевой пузырь

```css
background: rgba(255, 255, 255, 0.97);
border: 1px solid #d7e6ff;
border-radius: 18px;
box-shadow: 0 14px 32px rgba(56, 101, 171, 0.2);
color: #355a90;
```

Пузырь имеет ромбовидный хвост (CSS `::after`, поворот 45deg), направленный вниз к маскоту.

### Фоновое свечение

Вокруг маскота — `radial-gradient` свечение:

```css
background: radial-gradient(
	ellipse at center,
	rgba(74, 127, 212, 0.28) 0%,
	rgba(44, 90, 160, 0.12) 45%,
	transparent 70%
);
filter: blur(50px);
```

## 6. UI-стиль

### Карточки (Glass Card — основной паттерн)

```css
background: rgba(255, 255, 255, 0.7);
backdrop-filter: blur(8px);
border: 1px solid rgba(44, 90, 160, 0.1);
border-radius: 16–24px;
```

**Hover:** `translateY(-2..4px)` + `box-shadow: 0 4–6px 20–24px rgba(44, 90, 160, 0.12)`

### Кнопки

**Primary (pill):**

```css
background: linear-gradient(135deg, #2c5aa0, #4a7fd4);
color: #ffffff;
border-radius: 999px;
font-weight: 600;
box-shadow: 0 8px 30px rgba(44, 90, 160, 0.35);
```

**Secondary (glass pill):**

```css
background: rgba(255, 255, 255, 0.5);
backdrop-filter: blur(4px);
border: 1px solid rgba(44, 90, 160, 0.15);
border-radius: 999px;
color: #2c5aa0;
font-weight: 500;
```

### Бейджи / Теги

```css
background: rgba(44, 90, 160, 0.06);
color: #2c5aa0;
border-radius: 999px;
font-size: 12–14px;
font-weight: 500;
```

### Ключевые border-radius

| Значение  | Где                          |
| --------- | ---------------------------- |
| `999px`   | Кнопки, бейджи, теги, пилюли |
| `50%`     | Аватары, нумерованные кружки |
| `20–24px` | Карточки                     |
| `16–18px` | Малые карточки, пузыри       |
| `12px`    | Вложенные блоки              |

### Тени (система)

| Интенсивность | Значение                                                  |
| ------------- | --------------------------------------------------------- |
| **Subtle**    | `0 2px 12–20px rgba(44, 90, 160, 0.08–0.12)`              |
| **Medium**    | `0 4px 20–24px rgba(44, 90, 160, 0.15–0.25)`              |
| **Strong**    | `0 8px 30–40px rgba(44, 90, 160, 0.35–0.45)`              |
| **Glow**      | Двойная тень с добавлением `rgba(74, 127, 212, 0.05–0.2)` |

## 7. Tone of Voice

### Как бренд говорит

| Параметр            | Характеристика                                                             |
| ------------------- | -------------------------------------------------------------------------- |
| **Формальность**    | Средняя — уважительно, но без канцеляризма                                 |
| **Эмоциональность** | Тёплая — забота о пользователе, но без панибратства                        |
| **Экспертность**    | Присутствует — через структурированность и конкретику (цифры, этапы, роли) |
| **Юмор**            | Лёгкий — через маскота и его реплики, не через основной текст              |
| **Обращение**       | Через «вы» (уважительное), часто через вопросы («Кому подходит?»)          |

### Принципы текста

- Заголовки — короткие, утвердительные, конкретные
- Подзаголовки — раскрывают суть без воды
- Списки предпочтительнее абзацев
- Маскот говорит неформальнее, чем основной текст — это единственный источник «характера» в тексте
- Позитивные формулировки доминируют; негатив только в контексте «кому не подходит»

## 9. Анимации

### Основные паттерны (Framer Motion)

**fadeUp (универсальный):**

```js
hidden:  { opacity: 0, y: 20 }
visible: { opacity: 1, y: 0, transition: { duration: 0.5 } }
```

**stagger (контейнер):**

```js
hidden:  { opacity: 0 }
visible: { opacity: 1, transition: { staggerChildren: 0.15 } }
```

**slideFromRight (маскот):**

```js
initial:  { opacity: 0, x: 40 }
animate:  { opacity: 1, x: 0 }
transition: { duration: 0.7, delay: 0.3 }
```

**Scroll-driven (маскот фоновый):**

```js
useScroll({ target: sectionRef, offset: [...] })
useTransform(scrollYProgress, [0, 0.35], [1, 0])     // opacity
useTransform(scrollYProgress, [0, 0.35], [0, 100])    // y
useTransform(scrollYProgress, [0, 0.35], [1, 0.85])   // scale
```

### CSS-переходы

```css
transition:
	transform 0.2s ease,
	box-shadow 0.2s ease; /* карточки */
transition: all 0.3s ease; /* primary-кнопки */
transition: all 0.2s ease; /* навигация, secondary-кнопки */
transition: color 0.2s ease; /* ссылки */
```

**Триггер появления:** `useInView({ once: true, margin: '-100px' })`
